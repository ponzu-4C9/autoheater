## autoheaterBunkakaiWifiについて、
![IMG20260112154731](https://github.com/user-attachments/assets/91440b2a-1463-45b4-986d-6c0ae545c68f)

状態は['初期加熱','k測定','昇温(T1)','降下測定(L)','維持(T1)','昇温(T2)','維持(T2)','完了'];がある。熱効率K、放熱係数lについては、後述。

デューティ比とは0から1のdoubleであり、出力のことである。0は最小（まったく過熱しない）1は最大（最大火力）である。この値を適切な値に管理するのがこの制御プログラムの目標である。

この制御プログラムの中核は以下の数式である。

$$
\Huge　\frac{dT}{dt} = kx - l(T - T_{roomtemp})
$$

$T$:温度（℃）

$t$:プログラム開始からの時刻

$k$:画像にあった熱効率k。正の値である。

$x$:デューティ比。0から1のdouble

$l$:放熱係数l。上記の数式内では正の値であるが、プログラム中では負の値である。よって実際のロジック中では  $kx + l(T - T_{roomtemp})$ となる。わかりにくくて申し訳ない。

$T_{roomtemp}$:室温。20。
